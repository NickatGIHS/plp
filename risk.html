Risk Taker Goal

For my Risk Taker goal, I have decided to make a website for a comic that I started when I was younger, Silly Stik.
The site will be coded from scratch by me, in PHP (a Hypertext Preprocessor which is basically HTML with more features).

PLAN

I have been toying with the idea of creating a website for Silly Stik for a while now. I think that doing this for PLP has helped me feel like I can do it.

I, like with my Change Maker goal, really just went into this without much of a plan. Ince again, I just went with things as they came.

Some notes about the site format

At the start, I decided that ideally for the backend of the website, I would only have to add a file or two (the comic and data for the comic), and the website would add it automatically, as shown in this quick drawing.

I feel like I should also mention that the site format will probably be loosely based off xkcd's website.
